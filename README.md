# C++ para Iniciantes – 15 Exemplos Práticos

Este repositório contém 15 programas simples em C++ desenvolvidos para quem está começando a aprender programação. Cada arquivo aborda um conceito fundamental, como entrada/saída, estruturas de decisão, laços, vetores, matrizes, funções e manipulação de strings.

## Como compilar e executar

Para compilar qualquer um dos programas, utilize um compilador C++ como `g++` (no Linux/macOS) ou `cl` (no Windows com Visual Studio). Exemplo com `g++`:

```bash
g++ nome_do_arquivo.cpp -o nome_do_programa
./nome_do_programa      # Linux/macOS
nome_do_programa.exe    # Windows
